# Project Development Report

## 1. Project Analysis and Choices

### Project Goal

Create a Pac-Man clone.

The goal of the project was to develop a Pac-Man clone using Python.

### Main Choices

* **Choice 1:** Use of Pygame as the rendering library.
* **Choice 2:** Implementation of a manager system to share inputs and information across the application.
* **Choice 3:** Restructuring the maze generator output to create a data structure that is easier to use throughout the project.

## 4. Challenges and Conflicts

### Main Challenges

A complete restructuring of the project during the early development stages was required to make future implementations easier and more maintainable.

### Resolution

* The introduction of the manager system made data sharing across the application easier to handle and maintain.
* The decision to use a single ghost class instead of implementing four separate strategies simplified the management of pathfinding differences between ghosts.
